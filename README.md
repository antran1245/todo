# Objective

Allow user to create a todo list.

## Run Application
- npm start
- yarn start

## Technologies

**Languages**
- JavaScript
- CSS
- HTML

**Framework/Libraries**
- React
- React-bootstrap
- Express
- Mongoose

**Database**
- MongoDB

## Features

- Dark and Light mode by click of button (<span>&#9728;</span> / <span>&#x1F319;</span>)
    - Change in background image by mode.
- Remove button to remove item from list.
